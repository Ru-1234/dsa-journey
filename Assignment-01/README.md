# CodeCollab SLL (Question-01)

Singly Linked List based contributor queue management system developed for Data Structures Assignment 1.

## Features

- Mentor-priority contributor insertion
- Active contributor promotion
- Inactive contributor removal
- Consecutive collaborator merging
- Segment reversal using pointer manipulation
- Session analytics and reporting
- Menu-driven console interface

## Concepts Used

- Singly Linked Lists
- Dynamic memory allocation
- Pointer manipulation
- Node deletion and insertion
- Traversal algorithms
- Linked list reversal

## Constraints

- No STL containers
- No arrays for storage
- Iterative logic only
- Pointer-based operations
